1. Jak uruchomić aplikację backendową?

Po otworzeniu projektu należy wykonać następujące polecenia:
cd rxjs-chat
cd server
node index.js

2. Jak uruchomić aplikację w przeglądarce ?

Po otworzeniu projektu należy wykonać następujące polecenia:
cd rxjs-chat
cd client
ng serve

Następnie w przeglądarce należy użyć adresu: localhost:4200
